# mchat
mchat is a micro chat application

## usage
```sh
# clone
git clone https://github.com/subrat-lima/mchat
cd mchat

# install dependencies
uv sync

# initialize
. ./.venv/bin/activate
python init.py

# run application
python run.py
```

## tech stack
- backend: fastapi
- database: sqlite
- frontend: vanilla-js, picocss
